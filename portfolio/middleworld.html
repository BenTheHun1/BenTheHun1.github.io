A text-based RPG set on a disc-shaped, world-between-worlds, which opens up to the other realities of the multiverse with the passing of time.

This was the final project for my Narrative Design class. This was a solo project, so I was responsible for all aspects.

A full-length, dramatically expanded and improved version is currently in development.
